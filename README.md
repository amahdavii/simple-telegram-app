# Simple Telegram Webapp with Reactjs

• Chats List Page

• Chat Inner Page

• Search By Username

## Installation

Step 1 - For install node_module packages:

```bash
yarn
```

Step 2 -For starting JSON server on port 3002:

```bash
yarn server
```

Step 3 - For Running React app:

```bash
yarn start
```
